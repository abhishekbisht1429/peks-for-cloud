## Instructions to run project

### Build Project
- Install `cmake` on your system if not already installed.
- Run `cmake -S . -B cmake-build-debug`
- Run `cmake --build cmake-build-debug`

### Run Project
Run `cd cmake-build-debug`

#### Server command
`./server`

#### Data Owner command
`./data_owner`

#### Data Consumer command
`./data_consumer`
